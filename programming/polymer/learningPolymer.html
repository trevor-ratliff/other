This is a test of web components using the polymer.js library.   The custom element is a row repeater - it will allow you to duplicate whatever content is placed inside updating any ids with a row reference.

The row-repeat tag can be configured with 3 attributes:

  • contentid -- allows you to set the id of the elements content tag in the shadow dom
  • count -- sets the initial number of rows for the element
  • id -- sets the id for the element (important for internal references)
